# gtags

## 1.安装

1.1 Linux

1.1.1 源码编译安装

​    以安装6.6.5版本为例

​    a) 从官方网站（https://ftp.gnu.org/pub/gnu/global/）下载源码包

wget https://ftp.gnu.org/pub/gnu/global/global-6.6.5.tar.gz

​    b))解压

tar -xzvf global-6.6.5.tar.gz

​    c)进入源码目录

cd global-6.6.5

​    d)编译

./configure 
make

​    e)安装

sudo make install
